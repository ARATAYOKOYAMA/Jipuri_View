# [iOS Swift3]「じぷり」

## 概要

* 函館市にある陣川あさひ町会からの「イベント情報」「お知らせ」を簡単に見ることのできるアプリです。
* さらに、陣川あさひ町会が運営するイベントに対して「イベント参加申し込み」も可能です。
* MonacaとNCMBを用いて作成した「じぷり」をSwift3 verで作成したものです。
* セキュリティーの都合上NCMBのkeyは伏せてあります。

## 動作環境

* Xcode Version 8.3.2
* iPhone7(iOS 10.3)
